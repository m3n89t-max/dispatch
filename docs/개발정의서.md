# 제주 물류 배차 자동화 시스템 개발정의서 v1.0

---

# 1. 프로젝트 개요

## 1.1 목적
본 시스템은 제주 물류센터 가전/에어컨 배송·설치 배차를  
**실적 기반 + 전일 배차 + 자동화 + 공정성 확보** 구조로 개선하기 위한 시스템이다.

---

## 1.2 해결하려는 문제

현재 문제:

- 인력 기반 배차 → 불공정 발생
- 동일 4대 배차에도 완료 편차 발생
- 납기 변경으로 실제 수행량 변동
- 특정 팀/권역 배차 쏠림
- 안전/근무 관리 미흡

---

```text
